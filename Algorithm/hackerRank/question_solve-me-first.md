# Solve Me First
solveMeFirst 함수를 완성하여 두 정수의 합을 계산합니다.

### Example
a = 7  
b = 3  
Return 10

### 기능 설명
아래 편집기에서 solveMeFirst 함수를 완료하십시오.  
SolveMeFirst에는 다음과 같은 매개 변수가 있습니다.
* int a: 첫 번째 값
* int b: 두 번째 값

### Returns
- int: a와 b의 합

### 제약
1 ≤ a, b ≤ 1000

### 입력 예시
```
a = 2
b = 3
```

### 출력 예시
```
5
```

### 설명
2 + 3 = 5
